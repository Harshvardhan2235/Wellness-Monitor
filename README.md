
## 🚀 Usage

1. **Start the application** - Run `python app.py`
2. **Allow camera access** - Grant permission when prompted by browser
3. **Complete Daily Check-In** - Click "Daily Check-In" button in header
4. **Monitor wellness** - View real-time wellness score and suggestions
5. **Get recommendations** - Click "Get Personalized Suggestions" button
6. **Download reports** - Access daily CSV/JSON reports from dashboard

## 📊 How It Works

1. **Camera captures frames** every 2 seconds
2. **MediaPipe analyzes posture** (neck angle, shoulder tilt, back alignment)
3. **TensorFlow detects emotions** from facial expressions
4. **Wellness engine calculates score** based on detections + daily check-in
5. **AI generates 3 personalized suggestions** for improvement
6. **Dashboard updates** in real-time (every 10 seconds)
