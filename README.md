## 🌟 Overview

**Wellness Monitor** is an intelligent system that continuously evaluates your **physical posture** and **emotional state** using your webcam, and converts that into a **real-time wellness score**.

It combines **computer vision + machine learning + behavioral insights** to help users improve daily habits and maintain better mental & physical health.

---

## 🔥 Key Features

- 🎥 **Live Camera Monitoring** – Continuous frame capture every 2 seconds
- 🧍 **Posture Analysis** – Detects neck angle, shoulder tilt, back alignment using MediaPipe
- 🙂 **Emotion Detection** – Facial expression recognition using TensorFlow
- 📊 **Wellness Score Engine** – Combines posture + emotion + daily input
- 💡 **AI Suggestions** – 3 personalized improvement recommendations
- 📈 **Live Dashboard** – Updates every 10 seconds
- 📁 **Report Generation** – Export wellness data as CSV/JSON

## 🚀 Usage

1. Start the application:
   ```bash
   python app.py
   ```
2. Allow camera access in browser
3. Click **Daily Check-In**
4. Monitor your real-time wellness score
5. Click **Get Personalized Suggestions**
6. Download reports from dashboard

---

## 🧠 How It Works

1. 📸 Camera captures frames every 2 seconds
2. 🧍 MediaPipe analyzes posture (neck, shoulders, spine)
3. 🙂 TensorFlow detects emotions from facial expressions
4. 🧮 Wellness engine computes a score
5. 💡 AI generates personalized suggestions
6. 📊 Dashboard refreshes every 10 seconds
